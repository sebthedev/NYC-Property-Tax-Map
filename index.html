New York's property tax system is unfair. Similar properties are often taxed at wildly different rates. Millionaires' mansions are frequently taxed at lower rates than working-class apartments. Read more on Sidewalk Chorus about New York's flawed property tax system and how we can reform it.

This map was created by Sebastian Hallum Clarke, the author of Sidewalk Chorus. Countless experts and politicians have derided New York's unfair property tax system, but getting New York's legislators to act to fix the system first requires public awareness of the problems. This map is an attempt to help New Yorkers learn how their property tax system is failing them.

The code that powers this map is available on GitHub. All of the data that powers this map is from official City of New York sources. Read mode about the data, which was most recently updated on 27 October 2025 using data for the 2025-2026 tax year.
